## üìã Medium Description (200-300 words)

**Thorax: Autonomous AI Security Agent for Neo Blockchain**

Smart contract exploits cost the blockchain industry over $3 billion annually. Manual monitoring is impractical for 24/7 blockchain activity. **Thorax** solves this with an autonomous AI agent that never sleeps.

**How It Works:**

1. **Intelligent Analysis**: Users register Neo smart contracts. SpoonOS AI agents analyze the ABI, identifying dangerous methods and vulnerabilities. Google Gemini LLM generates risk scores (0-10) and identifies specific breach vectors.

2. **Autonomous Monitoring**: The agent continuously polls Neo RPC nodes, scanning blocks for contract events. No human intervention required‚Äîfully autonomous operation across multiple contracts.

3. **AI-Driven Detection**: Each event is analyzed by SpoonOS agents using Gemini LLM. The AI evaluates event patterns, determines if threats exist, classifies severity (low/medium/high/critical), and generates actionable recommendations.

4. **Automated Alerts**: Immediate email notifications to contract owners. Optional AI-generated voice alerts via ElevenLabs for critical events. Real-time dashboard with event streaming.

**Key Features:**
- ‚úÖ True autonomy with AI decision-making (not just automation)
- ‚úÖ SpoonOS AI Agent Framework integration (required for hackathon)
- ‚úÖ Native Neo N3 blockchain support (testnet/mainnet)
- ‚úÖ Multi-modal alerts (email + voice synthesis)
- ‚úÖ Legal compliance analysis
- ‚úÖ Production-ready implementation

**Use Cases:** DeFi protocol security, NFT marketplace protection, DAO governance oversight, token contract auditing.

**Tech Stack:** SpoonOS, Google Gemini, Neo N3, Next.js 16, FastAPI, PostgreSQL, ElevenLabs.

Thorax represents the future of Web3 security: autonomous, intelligent, and proactive.
